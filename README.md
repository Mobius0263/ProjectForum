Clone repository:
```
git clone https://github.com/Mobius0263/ProjectForum.git
```

In MAIN, open git bash here:
```
code .
```

Open Terminal in VSCode:
```
python -m pip install -r requirements.txt
```

Run server:
```
python manage.py runserver
```

Admin site:
```
http://127.0.0.1:8000/admin/
```
